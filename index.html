Online Drumkit

Welcome to my Online Drumkit! This is a coding project where I have used HTML, CSS and JavaScript in order to make a webpage that produces different sounds depending on which key/button is pressed while the page is open. I've made a simple design, to mirror the numpad that the sounds are bound to, and so to make use of the site easier. In doing so I have practiced with line-customisation (customising the <hr> tag and creating a class for a vertical line) as well as JavaScript as a whole.



Please, scroll down and enjoy the site!
